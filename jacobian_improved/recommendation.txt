Best strategy: robust_log / platt

To use in jacobian_eval.py, add after J is loaded:

  J_log = np.sign(J) * np.log1p(np.abs(J))
  J_rl = np.zeros_like(J_log)
  for i in range(J_log.shape[0]):
      row = J_log[i]
      med_r = np.median(row)
      iqr_r = np.percentile(row,75) - np.percentile(row,25)
      J_rl[i] = (row - med_r) / (iqr_r + 1e-9)
  J = J_rl

Then apply Platt calibration before score extraction.
